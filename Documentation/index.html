Structure des dossiers





Mise en place du FTP

  • Ouvrez votre client FTP et connectez-vous sur votre hébergeur.
  • Rendez-vous dans le dossier de votre projet.
  • Uploadez les fichiers de la maquette WebShop.


Structure des pages



  • index.html pour la première page d'accueil
  • index-2.html pour la seconde page d'accueil
  • shop.html pour la page Magasin
  • shop-left-sidebar.html pour la page Magasin avec une Sidebar à gauche
  • shop-right-sidebar.html pour la page Magasin avec une Sidebar à droite
  • shop-list-fullwidth.html pour la page Magasin en liste
  • shop-list-left-sidebar.html pour la page Magasin en liste avec une Sidebar à gauche
  • shop-list-right-sidebar.html pour la page Magasin en liste avec une Sidebar à droite
  • single-product.html pour la fiche produit détaillée
  • single-product-group.html pour la fiche produit détaillée groupée
  • single-product-variable.html pour la fiche produit détaillée avec choix personnalisés
  • single-product-sale.html pour la fiche produit détaillée promo
  • single-product-sticky.html pour la fiche produit détaillée (photos alternatives)
  • single-product-affiliate.html pour la fiche produit détaillée (pas de quantité à définir)
  • login-register.html pour la connection ou l'inscription
  • my-account.html pour la page "mon compte"
  • wishlist.html pour la page souhait d'achat
  • checkout.html pour la page du paiement
  • cart.html pour la page du panier
  • compare.html pour la page de comparaison de deux produits
  • about.html pour la page "À propos"
  • faq.html pour la page FAQ
  • 404.html pour la page erreur 404
  • contact.html pour la page contact

Structures des fichiers



Structure d'appel des Google Fonts

Pensez à changer les fonts dans le fichier style.css

          <!--All Google Fonts-->
          @import
          url('https://fonts.googleapis.com/css2?family=Rubik:ital,wght@0,300;0,400;0,500;0,600;0,700;1,300;1,400;1,500;1,600;1,700&display=swap');


Structure d'appel du CSS

Chaques fichiers ou ressources pour le CSS se trouve dans la section Head.

          <!-- ::::::::::::::All CSS Files here :::::::::::::: -->

          <link rel="stylesheet" href="assets/css/vendor/font-awesome.min.css">
          <link rel="stylesheet" href="assets/css/vendor/Pe-icon-7-stroke.css">
          <link rel="stylesheet" href="assets/css/plugins/animate.css">
          <link rel="stylesheet" href="assets/css/plugins/jquery-ui.min.css">
          <link rel="stylesheet" href="assets/css/plugins/swiper-bundle.min.css">
          <link rel="stylesheet" href="assets/css/plugins/nice-select.css">
          <link rel="stylesheet" href="assets/css/plugins/magnific-popup.min.css">
          <link rel="stylesheet" href="assets/css/plugins/ion.rangeSlider.min.css">

          <!-- Style CSS File -->
          <link rel="stylesheet" href="assets/css/style.css">


Structure d'appel du JavaScript

Chaques fichiers ou ressources pour le JavaScript se trouve à la fin de la section Body.

          <!-- ::::::::::::::All Javascripts here ::::::::::::::-->

          <!-- Vendor JS Files -->
          <script src="assets/js/vendor/bootstrap.bundle.min.js"></script>
          <script src="assets/js/vendor/jquery-3.6.0.min.js"></script>
          <script src="assets/js/vendor/jquery-migrate-3.3.2.min.js"></script>
          <script src="assets/js/vendor/modernizr-3.11.2.min.js"></script>

          <!-- Plugins JS Files -->
          <script src="assets/js/plugins/wow.min.js"></script>
          <script src="assets/js/plugins/jquery-ui.min.js"></script>
          <script src="assets/js/plugins/swiper-bundle.min.js"></script>
          <script src="assets/js/plugins/jquery.nice-select.js"></script>
          <script src="assets/js/plugins/parallax.min.js"></script>
          <script src="assets/js/plugins/jquery.magnific-popup.min.js"></script>
          <script src="assets/js/plugins/tippy.min.js"></script>
          <script src="assets/js/plugins/ion.rangeSlider.min.js"></script>

          <!-- Main js file that contents all jQuery plugins activation. -->
          <script src="assets/js/main.js"></script>
